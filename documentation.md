# Документация главной страницы AppleStore

## 1. Шапка сайта (Header)

### 1.1. Логотип AppleStore
**Расположение:** Верхний левый угол страницы  
**Внешний вид:** Состоит из двух элементов:
1. **Графический логотип** - стилизованное изображение яблока в виде SVG-иконки белого цвета
2. **Текстовое название** - слово "AppleStore" белого цвета, выполненное жирным шрифтом

**Функциональность:**
- Является кликабельной ссылкой
- При нажатии ведет на главную страницу (в текущей реализации перезагружает страницу)
- При скролле страницы уменьшается в размерах для экономии пространства
- При открытии мобильного меню полностью скрывается с плавной анимацией

### 1.2. Навигационная панель (для десктопов)
**Расположение:** По центру шапки между логотипом и правыми элементами
**Внешний вид:** Горизонтальное меню с текстовыми ссылками

**Элементы меню:**
- Каталог (ссылка-якорь к разделу категорий)
- iPhone (ссылка на соответствующий раздел)
- iPad (ссылка на соответствующий раздел)
- MacBook (ссылка на соответствующий раздел)
- Аксессуары (ссылка на соответствующий раздел)
- Поддержка (ссылка на раздел поддержки)

**Интерактивность:**
- Белый цвет текста на фоне шапки
- При наведении:
  - Текст меняет цвет на #287370
  - Снизу появляется анимированная белая линия, которая плавно заполняет всю ширину ссылки
- Все ссылки ведут на соответствующие разделы сайта

### 1.3. Бургер-меню (для мобильных устройств)
**Расположение:** Правый верхний угол шапки
**Внешний вид:** Три горизонтальные линии (гамбургер-иконка)

**Поведение:**
- Отображается только на экранах шириной до 900px
- При нажатии:
  - Иконка и логотип плавно исчезают
  - Справа появляется мобильное меню
  - Фон затемняется полупрозрачным наложением
  - Скролл основной страницы блокируется
- **НОВОЕ:** При открытии меню кнопка полностью скрывается (не превращается в крестик)

### 1.4. Иконка пользователя (только для десктопов)
**Расположение:** Верхний правый угол страницы  
**Внешний вид:** Иконка силуэта человека из библиотеки Font Awesome

**Функциональность:**
- Отображается только на экранах шириной более 900px
- Является кликабельной ссылкой
- Предназначена для перехода в личный кабинет или на страницу авторизации
- При наведении курсора:
  - Плавно увеличивается на 10%
  - Меняет цвет с белого на #287370 (более темный оттенок основного цвета)
- При нажатии открывает соответствующий раздел сайта

## 2. Мобильное меню

### 2.1. Структура меню
**Расположение:** Выдвигается справа, занимает 85% ширины экрана (до 400px максимум)

**Заголовок меню:**
- Текст "Меню" белого цвета
- Фон основного цвета (#308582)
- Кнопка закрытия (крестик) в правом верхнем углу

### 2.2. Кнопка профиля
**Расположение:** Верхняя часть меню, сразу под заголовком
**Внешний вид:**
- Иконка пользователя слева
- Текст "Мой профиль"
- Светло-серый фон (#f8f9fa)
- Нижняя серая граница

**Функциональность:**
- Кликабельная ссылка
- При наведении фон становится немного темнее (#eaeaea)
- Веет на страницу личного кабинета или авторизации

### 2.3. Навигационные ссылки
**Список разделов:**
1. Каталог (ссылка-якорь к разделу категорий)
2. iPhone (ссылка на соответствующий раздел)
3. iPad (ссылка на соответствующий раздел)
4. MacBook (ссылка на соответствующий раздел)
5. Аксессуары (ссылка на соответствующий раздел)
6. Поддержка (ссылка на раздел поддержки)
7. Контакты (ссылка на раздел контактов)
8. О магазине (ссылка на раздел "О нас")

**Визуальные характеристики:**
- Темный текст (#1d2029) на белом фоне
- Каждый пункт разделен тонкой серой линией
- При наведении:
  - Фон становится светло-серым (#f8f9fa)
  - Текст меняет цвет на #308582
  - Происходит плавный сдвиг текста вправо на 5px

### 2.4. Кнопка закрытия меню
**Расположение:** Верхний правый угол мобильного меню
**Внешний вид:** Белый крестик (иконка "×")

**Функциональность:**
- При нажатии:
  - Меню плавно скрывается вправо
  - Затемнение фона исчезает
  - Логотип и бургер-кнопка плавно появляются
  - Восстанавливается скролл страницы
- При наведении плавно увеличивается на 10%

### 2.5. Затемнение фона
**Внешний вид:** Полупрозрачный черный слой поверх всей страницы
**Функциональность:**
- Появляется при открытии мобильного меню
- Закрывает мобильное меню при клике в любую область вне меню
- Плавно исчезает при закрытии меню

## 3. Особенности навигации

### 3.1. Адаптивное переключение
**Переход между режимами:**
- **До 900px:** Отображается бургер-меню, десктопная навигация скрыта
- **901px и более:** Отображается десктопная навигация, мобильное меню недоступно

### 3.2. Закрытие меню
**Способы закрытия мобильного меню:**
1. Нажатие на крестик в меню
2. Нажатие на затемненную область вне меню
3. Нажатие на любую ссылку внутри меню
4. Нажатие клавиши Escape на клавиатуре

### 3.3. Доступность
**ARIA-атрибуты:**
- Бургер-кнопка имеет `aria-label="Открыть меню"`
- При открытии меню атрибут меняется на `aria-expanded="true"`
- При закрытии - на `aria-expanded="false"`

## 4. Адаптивные изменения

### 4.1. На мобильных устройствах (до 480px)
- Логотип в шапке уменьшается
- Заголовок в герой-секции становится меньше
- Карточки категорий располагаются вертикально
- Подвал меняется с трехколоночной на одноколоночную структуру
- Юридические ссылки располагаются вертикально
- **НОВОЕ:** Ширина мобильного меню увеличивается до 90% экрана

### 4.2. На планшетах (481px-900px)
- Десктопная навигация скрывается
- Появляется бургер-меню
- Карточки категорий: 2 в первом ряду, 1 во втором
- Подвал: две колонки (первая занимает 100% ширины, вторая и третья - по 50%)
- Уменьшаются отступы и размеры шрифтов

### 4.3. На десктопах (901px+)
- Полная трехколоночная структура
- Максимальные размеры шрифтов
- Отображается горизонтальная навигационная панель
- Отображается иконка пользователя в шапке
- Все элементы отображаются в соответствии с дизайном
- **НОВОЕ:** Мобильное меню полностью скрыто и недоступно

## 5. Интерактивные эффекты

### 5.1. Эффект поднятия (Lift Effect)
Применяется к:
- Карточкам категорий при наведении
- Кнопке в герой-секции при наведении
- **НОВОЕ:** Кнопке закрытия мобильного меню при наведении

**Реализация:** Плавное перемещение элемента вверх на 3-10 пикселей с одновременным увеличением тени

### 5.2. Эффект изменения цвета (Color Transition)
Применяется к:
- Всем ссылкам и кнопкам при наведении
- Иконке пользователя в шапке
- **НОВОЕ:** Ссылкам в мобильном меню

**Реализация:** Плавное изменение цвета за 0.3 секунды

### 5.3. Эффект появления (Fade-in Effect)
Применяется к:
- Карточкам категорий при прокрутке страницы
- **НОВОЕ:** Логотипу и бургер-кнопке при закрытии мобильного меню

**Реализация:** Плавное изменение прозрачности и видимости элемента

### 5.4. Эффект уменьшения шапки (Header Shrink)
**Реализация:** При прокрутке страницы более чем на 50 пикселей шапка уменьшает высоту и размер логотипа для экономии места

### 5.5. Эффект выдвижения (Slide-in Effect)
**НОВОЕ:** Применяется к мобильному меню
**Реализация:** Плавное выдвижение меню справа за 0.4 секунды
