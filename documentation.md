# Документация главной страницы AppleStore

## 1. Шапка сайта (Header)

### 1.1. Логотип AppleStore
**Расположение:** Верхний левый угол страницы  
**Внешний вид:** Состоит из двух элементов:
1. **Графический логотип** - стилизованное изображение яблока в виде SVG-иконки белого цвета
2. **Текстовое название** - слово "AppleStore" белого цвета, выполненное жирным шрифтом

**Функциональность:**
- Является кликабельной ссылкой
- При нажатии ведет на главную страницу (в текущей реализации перезагружает страницу)
- При скролле страницы уменьшается в размерах для экономии пространства

### 1.2. Иконка пользователя
**Расположение:** Верхний правый угол страницы  
**Внешний вид:** Иконка силуэта человека из библиотеки Font Awesome

**Функциональность:**
- Является кликабельной ссылкой
- Предназначена для перехода в личный кабинет или на страницу авторизации
- При наведении курсора:
  - Плавно увеличивается на 10%
  - Меняет цвет с белого на #287370 (более темный оттенок основного цвета)
- При нажатии открывает соответствующий раздел сайта

## 2. Герой-секция (Hero Section)

### 2.1. Фоновое изображение
**Внешний вид:** Крупное качественное фото продуктов Apple в минималистичной обстановке
**Особенности:**
- Наложен полупрозрачный темный градиент для улучшения читаемости текста
- Занимает всю ширину экрана
- Масштабируется и обрезается для разных размеров экрана

### 2.2. Основной заголовок
**Текст:** "Техника Apple. Эффективность и стиль."
**Визуальные характеристики:**
- Самый крупный текст на странице (3rem)
- Белый цвет для контраста с темным фоном
- Жирное начертание
- Расположен по центру секции

### 2.3. Подзаголовок
**Текст:** "Инновационные устройства, которые меняют ваш подход к работе, творчеству и повседневной жизни."
**Визуальные характеристики:**
- Средний размер шрифта (1.2rem)
- Белый цвет с небольшой прозрачностью (90%)
- Расположен под основным заголовком
- Ограничен по ширине для лучшей читаемости

### 2.4. Кнопка "Смотреть каталог"
**Внешний вид:**
- Прямоугольная форма со скругленными углами (30px)
- Прозрачный фон с белой рамкой толщиной 2px
- Белый текст средней жирности

**Интерактивность:**
- При наведении курсора:
  - Фон заполняется основным цветом (#308582)
  - Рамка меняет цвет на #308582
  - Текст остается белым
  - Кнопка плавно поднимается на 3 пикселя вверх
- При нажатии прокручивает страницу к разделу с категориями товаров

## 3. Раздел категорий товаров

### 3.1. Заголовок раздела
**Текст:** "Категории товаров"
**Визуальные характеристики:**
- Крупный текст (2.2rem)
- Темный цвет (#1d2029)
- Расположен по центру над карточками категорий
- Жирное начертание

### 3.2. Карточка категории iPhone
**Компоненты:**
1. **Фоновое изображение** - фотография iPhone на контрастном фоне
2. **Наложение** - полупрозрачное темное затемнение для улучшения читаемости текста
3. **Название категории** - слово "iPhone" белым цветом крупным шрифтом
4. **Кнопка "Смотреть все"** - белая рамка с текстом внутри

**Интерактивность:**
- При наведении на всю карточку:
  - Карточка плавно поднимается на 10 пикселей вверх
  - Тень становится более выраженной
  - Затемнение фона становится менее интенсивным
  - Кнопка меняет цвет: фон становится #308582, рамка исчезает
- Вся карточка является кликабельной ссылкой

### 3.3. Карточка категории iPad
**Аналогична карточке iPhone, но содержит:**
- Фоновое изображение iPad
- Название "iPad"
- Та же кнопка "Смотреть все"

### 3.4. Карточка категории MacBook
**Аналогична предыдущим, но содержит:**
- Фоновое изображение MacBook
- Название "MacBook"
- Та же кнопка "Смотреть все"

**Расположение карточек:**
- На десктопах: 3 карточки в ряд с равными отступами
- На планшетах: 2 карточки в первом ряду, 1 во втором
- На мобильных: все карточки в одну колонку

## 4. Подвал сайта (Footer)

### 4.1. Верхняя часть подвала (3 колонки)

**Колонка 1: О магазине**
- **Мини-логотип** - уменьшенная версия логотипа из шапки
- **Название магазина** - "AppleStore" белым цветом
- **Описание магазина** - краткий текст о том, что это официальный дилер Apple в России

**Колонка 2: Контакты**
- **Заголовок** - слово "Контакты" белым цветом
- **Телефон** - номер +7 (800) 123-45-67 с иконкой телефона, кликабельная ссылка (при нажатии предлагает позвонить)
- **Email** - адрес info@apple-store.ru с иконкой конверта, кликабельная ссылка (при нажатии открывает почтовый клиент)
- **Адрес** - физический адрес магазина с иконкой метки

**Колонка 3: Быстрые ссылки**
- **Заголовок** - слово "Разделы" белым цветом
- **Ссылка на iPhone** - с иконкой телефона
- **Ссылка на iPad** - с иконкой планшета
- **Ссылка на MacBook** - с иконкой ноутбука
- **Ссылка на аксессуары** - с иконкой наушников

**Интерактивность ссылок в подвале:**
- При наведении меняют цвет с белого на #287370
- Сохраняют плавность перехода

### 4.2. Нижняя часть подвала

**Разделительная линия**
- Тонкая горизонтальная линия (1px)
- Цвет: белый с прозрачностью 20%
- Расположена между колонками и копирайтом

**Информация о правах**
- Текст: "© 2026 AppleStore. Все права защищены."
- Серый цвет с прозрачностью 80%
- Расположен по центру

**Юридические ссылки**
- "Политика конфиденциальности" - ссылка на соответствующий документ
- "Пользовательское соглашение" - ссылка на соответствующий документ
- При наведении становятся полностью белыми (увеличивается непрозрачность)

## 5. Адаптивные изменения

### 5.1. На мобильных устройствах (до 480px)
- Логотип в шапке уменьшается
- Заголовок в герой-секции становится меньше
- Карточки категорий располагаются вертикально
- Подвал меняется с трехколоночной на одноколоночную структуру
- Юридические ссылки располагаются вертикально

### 5.2. На планшетах (481px-768px)
- Карточки категорий: 2 в первом ряду, 1 во втором
- Подвал: две колонки (первая занимает 100% ширины, вторая и третья - по 50%)
- Уменьшаются отступы и размеры шрифтов

### 5.3. На десктопах (769px+)
- Полная трехколоночная структура
- Максимальные размеры шрифтов
- Все элементы отображаются в соответствии с дизайном
